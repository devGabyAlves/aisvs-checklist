# AISVS Checklist Interativo

Página web interativa para explorar os controles do [OWASP AISVS](https://github.com/OWASP/AISVS) por nível de verificação.

## Visão geral

Este projeto consome os capítulos oficiais do AISVS 1.0 diretamente do GitHub, organiza os controles em formato de checklist e permite acompanhar o progresso de implementação com persistência local.
